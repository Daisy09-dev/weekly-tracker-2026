# daisy.track 🌱
### Weekly Habit Tracker — April to June 2026

A personal habit tracking web app built to reduce chaos, build consistency, and stay in control — until June 2026 and beyond.

---

## ✨ Features

- **Daily Tracker** — Check off 8 core habits every day, rate energy/focus/mood (1–5), and write a one-line reflection
- **Fail-Safe Mode** — When overwhelmed, switch to just 3 core habits (sleep, spiritual, study) so you never fully collapse
- **Live Score** — Today's completion percentage updates in real time
- **Weekly View** — See all 7 days at a glance, total check count, days shown up, and your current phase badge
- **Sunday Reflection** — 4 guided questions to review your week honestly
- **Monthly Checkpoint** — 4 animated progress rings (sleep, study, phone, mood) calculated from your real data
- **Monthly Check-in** — 4 yes/no questions to track your growth month over month
- **Persistent Storage** — All data saves to `localStorage` — it stays between sessions
- **Phase Tracker** — Automatically shows your current growth phase (Phase 1–4) based on week number

## 🧠 The System

| Phase | Weeks | Focus | Goal |
|-------|-------|-------|------|
| 🟢 Phase 1 | 1–2 | Showing up | 50–60% consistency |
| 🟡 Phase 2 | 3–6 | Stability | 60–75% consistency |
| 🔵 Phase 3 | 7–12 | Control | 75–85% consistency |
| 🟣 Phase 4 | Before June | Refinement | Consistent identity |

### Weekly Score System
- 🟢 80%+ → Strong week
- 🟡 60–79% → Improving
- 🔴 Below 60% → Adjust the system, not yourself

### The 8 Daily Habits
1. Slept before 11:00 PM
2. Phone away by 10:30 PM
3. Spiritual (5–10 min)
4. Controlled Phone Use
5. 2 Study Blocks (25 min each)
6. 1 Assignment Progress
7. Physical Movement (5–10 min)
8. 1-Line Reflection written

### Fail-Safe Mode (3 habits only)
- Sleep on time
- 1 Study Block
- Spiritual

## 🚀 How to Use

### Option 1 — Open Locally
1. Download or clone this repo
2. Open `index.html` in any browser
3. Start tracking!

### Option 2 — Deploy with GitHub Pages
1. Fork or push this repo to your GitHub account
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your live tracker will be at `https://yourusername.github.io/weekly-tracker-2026`

---

## 💾 Data & Privacy
All data is stored locally in your browser using `localStorage`. Nothing is sent to any server. Your reflections, habits, and scores are 100% private.
